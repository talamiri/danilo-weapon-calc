DANYEL - Static FiveM Weapon Calculator
---------------------------------------

Files:
- index.html  (open in browser directly)

How to use:
1. Download and unzip.
2. Open index.html in your browser (double-click or drag into browser).
3. Edit quantities and add stock as needed.

To host online:
- Upload to GitHub Pages (push to repo and enable Pages).
- Or drop into Netlify/Vercel (drag & drop for static site or connect repo).
